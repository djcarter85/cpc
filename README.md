# Cambridgeshire Planting Collective website

This repository contains the source for [the website of Cambridgeshire Planting
Collective](https://cgpcollective.com/).

## Development

To work on this repo, clone it and run `bundle exec jekyll serve` from your
terminal.

## Architecture

This site is built using Jekyll. It uses Bulma CSS for styling, compiled using
SASS.

The site is hosted using GitHub Pages, deployed from the `master` branch.

There are DNS records configured to hook up the domain name to the deployed
site.
